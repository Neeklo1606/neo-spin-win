# N2O Admin Panel

Admin panel for N2O Delivery Service.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=http://localhost:3000
```

3. Start dev server:
```bash
npm run dev
```
